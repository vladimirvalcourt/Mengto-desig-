01 / Destination

A destination, not a route.

Define arrival with enough precision that the system can choose a better path than the one you would have prescribed.

Goal before steps

02 / The fence

Rules make freedom safe.

A short set of laws protects the work while leaving the route open: no invented proof, no silent spending, no irreversible action without a gate.

Five laws, wide runway

03 / The bar

Good is not measurable.

Replace adjectives with a test: can a first-time visitor explain the story, find the action, and move through every seam without noticing it?

Evidence over confidence

04 / The loop

Close the largest gap.

Build, verify against the bar, repair the biggest failure, then repeat. Progress is a loop with a finish line, not a pile of activity.

Build → verify → repair

05 / The route

Spend depth where it compounds.

Use the strongest judgment on foundations and the lighter machinery on repetition. The route is a budget, not a reflex.

Route work by consequence

06 / Independent proof

The builder never grades itself.

Use a fresh verifier with the real output and one job: find what still breaks the contract. Finish when the evidence survives that pass.

A skeptic with no excuses

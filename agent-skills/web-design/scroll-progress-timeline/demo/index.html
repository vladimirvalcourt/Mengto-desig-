A process with evidence

From question to proof.

Four phases connect a loose possibility to something observable. The rail records where the reader is—nothing more, nothing hidden.

Field method / 01

Build what the evidence can hold.

01/ 04 active phase
  1. 01 / Frame

    Ask the useful question.

    Turn an open ambition into one observable tension, one audience, and one decision the work must support.

    Output — written premise
    Question field / unresolved
  2. 02 / Observe

    Collect the signal.

    Look for repeated behavior, exceptions, workarounds, and language. Keep source evidence separate from interpretation.

    Output — evidence ledger
    Signal map / 12 observations
  3. 03 / Shape

    Make the smallest system.

    Connect only the states required to test the idea. Give each transition a visible cause and a reversible path.

    Output — working model
    State model / three transitions
  4. 04 / Verify

    Let reality answer.

    Observe the result against the premise, record the gap, and keep the next decision narrower than the one before it.

    Output — verified finding
    Decision record / complete

A line connects the order. Evidence gives the order meaning.
